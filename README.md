# CodeAlpha Python Programming Internship - Stock Portfolio Tracker Task 2

## 📌 Project Description

This is a simple Stock Portfolio Tracker built using Python. It allows users to input stock names and quantities, calculates total investment value based on predefined prices, and optionally saves the result to a file.

## 🎯 Features

* User input for stock names and quantities
* Predefined stock price dictionary
* Calculates total investment value
* Saves output to a `.txt` file

## 🛠️ Technologies Used

* Python
* Basic concepts: dictionary, loops, input/output, file handling

## ▶️ How to Run

1. Install Python
2. Open terminal
3. Run:

   ```bash
   python stock_portfolio.py
   ```

## 📂 Output

* Displays investment details in terminal
* Optionally saves data in `portfolio.txt`

## 📊 Example

```
AAPL: 2 shares × 180 = 360
TSLA: 1 shares × 250 = 250

Total Investment: 610
```
